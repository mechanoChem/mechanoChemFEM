Detailed Description

template<int dim>
class hpFEM< dim >

This class is intended to be base/abstract class. It contains three essential objects in deal.ii: Triangulation<dim>, hp::DoFHandler<dim> and ConstraintMatrix. Those three will be shared in other classes:

solveClass< dim, matrixType, vectorType > and FEMdata< dim, vectorType >
